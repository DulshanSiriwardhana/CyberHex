# CyberHex v4.0 — Release No. 01: *The Word*

## Design intent

v4.0 begins with a single atomic unit: **the brand word** `CyberHex`. Every surface, release, and theme variant hangs off this typographic lockup. Release **No. 01** is the first serial in the v4 lineage — like firmware stamped on hardware.

| Principle | v3.0 | v4.0 No. 01 |
|-----------|------|-------------|
| Brand anchor | Scattered cyan/green accents | One wordmark system |
| Versioning | `v3.0` string in badge | Serial `01` + codename *The Word* |
| Typography | Inter only | **Outfit** wordmark + Inter UI |
| Accent | Generic cyan | **Signal cyan** + **hex gold** underline |

## The wordmark lockup

```
     01          ← serial stamp (mono, −90°, left rail)
  CyberHex       ← CYBER (white, tracked) + Hex (signal cyan)
  ───────        ← gradient underline: cyan → gold → violet
```

- **Cyber** — neutral, uppercase tracking (`0.14em`), weight 800  
- **Hex** — `--v4-signal-cyan` with tri-tone underline (hex pun: cyan + gold + violet)  
- **01** — release index, not a version number; future releases become No. 02, No. 03
